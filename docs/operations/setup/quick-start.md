## 起動

```bash
npm run dev
```

ブラウザで **http://localhost:3000** を開く。

## 初回セットアップ

1. **ソース設定** — 収集対象URLを追加
2. **配信設定** — 配信時刻・受信メール・0件時動作・コスト上限を設定
3. **メール送信テスト** — `/api/test-email` または設定画面からテスト

詳細は **docs/QUICK_START.md** を参照。
